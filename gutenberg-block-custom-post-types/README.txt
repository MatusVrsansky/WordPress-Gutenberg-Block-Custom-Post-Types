== Description ==

Plugin to register custom Gutenberg Block on displaying registered Custom Post Types.
User can set, which Custom Post Types will be displayed and also can set how many posts
will be displayed on the page.

== Requirements ==

This plugin works only when these plugins are activated -> Meta Box, Meta Box AIO
